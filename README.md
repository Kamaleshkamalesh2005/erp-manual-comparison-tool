## üõ†Ô∏è Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/Kamaleshkamalesh2005/erp-manual-comparison-tool.git
    cd erp-manual-comparison-tool
    ```
2. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

### Running Locally

```bash
streamlit run app.py
```
App will launch and open in your browser (`http://localhost:8501` by default).

---

## üìÑ Usage

1. **Upload Files:** Upload two (or more) Excel/CSV files for comparison.
2. **Set Criteria:** Select columns or fields to use for matching and comparison.
3. **Review Results:** View highlighted discrepancies, download reports, and tag reviewed data.
4. **Export:** Download the final comparison report if needed.
